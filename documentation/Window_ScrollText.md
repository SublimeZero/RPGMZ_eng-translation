# Class: Window_ScrollText

## Superclass: [Window_Base](Window_Base.md)

A window that displays [scrolling text] across the entire screen.

The text to be displayed is set using the methods [setScroll()](Game_Message.md#setscroll-speed-nofast) and [add()](Game_Message.md#add-text) from the [$gameMessage](global.md#gamemessage-game_message), not within this Window_ScrollText itself.

Changes were made in versions v1.1.1, v1.2.0, and v1.7.0.

Related classes: [Scene_Map](Scene_Map.md), [Scene_Battle](Scene_Battle.md), [Game_Message](Game_Message.md)
